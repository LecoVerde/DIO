Surgimento da Internet

(Voltar)

Tim Berners-Lee é conhecido por inventar a World Wide Web (WWW) enquanto trabalhava no CERN, o laboratório europeu de física de partículas, no final dos anos 1980. Ele desenvolveu um sistema que permitia a organização e a interconexão de informações através de hipertexto e links, tornando possível navegar e acessar documentos de forma eficiente na internet. Berners-Lee também criou o primeiro navegador web, servidor web e as especificações para URLs, HTTP e HTML, fundamentais para a expansão e democratização da internet globalmente.

Como Funciona o HTML

(Voltar)

HTML é a linguagem usada para estruturar documentos na web. Utiliza tags para marcar diferentes partes do conteúdo, como cabeçalhos, parágrafos e imagens. As tags têm atributos que definem características adicionais dos elementos. Os navegadores interpretam o HTML para renderizar as páginas da web conforme o código é escrito, permitindo uma apresentação consistente e acessível do conteúdo online.

O Que São "Tags"?

(Voltar)

Atributos em HTML são informações adicionais que podem ser incluídas dentro das tags para especificar detalhes sobre como um elemento deve ser apresentado ou comportar-se. Eles permitem personalizar características como a fonte de uma imagem (src em "< img >"), o destino de um link (href em "< a >"), entre outros, melhorando a funcionalidade e a aparência das páginas web.
